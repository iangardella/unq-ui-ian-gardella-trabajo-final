# TP-Final-Ian-Gardella

## Instalacion
Una vez cloneado el repositorio se debera situarse en la Carpeta de unq-ui-ian-gardella-trabajo-final

cd unq-ui-ian-gardella-trabajo-final

Instalar todas las dependencias con:

npm install

Y luego ejecutar 

npm run dev 

Para poder levantar la aplicacion web
