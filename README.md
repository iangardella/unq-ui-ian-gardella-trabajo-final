# unq-ui-ian-gardella-trabajo-final

*Trabajo Final Integrador de la materia Construcción de Interfaces de Usuario.*

## Instrucciones de instalación y ejecución

Para correr el proyecto es necesario clonar el repositorio de forma local. Luego, ejecutar los siguientes comandos en el orden indicado:

1. **Moverse a la carpeta del proyecto**

```bash
cd unq-ui-ian-gardella-trabajo-final
```
2. **Instalar todas las dependencias**
npm install
```bash
npm install
```
3. **Iniciar script de Dev para empezar el servidor de desarrollo local**
```bash
npm run dev
```
Una vez iniciado el servidor ingresa al url dada y al port indicado por pagina web
